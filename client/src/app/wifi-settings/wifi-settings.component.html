Wifi Settings
You have an Ethernet connection, but your NVR is connected to the Cloud service through Wi-Fi. To make changes to the Wi-Fi settings, this connection must be through Ethernet. To switch the connection to Ethernet, click the Reset Connection button, then wait until display shows the current Wi-Fi status again.
You have no Ethernet connection available, To make changes to the Wi-Fi settings, you must connect to the NVR through Ethernet. To switch the connection to Ethernet, connect an Ethernet cable between your router and the NVR then select Get Active Local IP address(es) to find the IP address for the Ethernet connection. Enter this address into your browser (which should be on the same subnet as the NVR), and return to WiFi Admin -> WiFi Settings.
